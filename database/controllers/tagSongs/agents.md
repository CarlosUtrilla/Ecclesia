# TagSongs Controller Agent

## Descripción

Controlador para la gestión de tags y su relación con canciones.

## Responsabilidad

- CRUD de tags.
- Asignación y consulta de tags en canciones.
- El campo `shortCut` se mantiene solo por compatibilidad de esquema; se autogenera internamente y no se usa como funcionalidad de atajos.

## Ubicación

`database/controllers/tagSongs/`
